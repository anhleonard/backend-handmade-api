To install open terminal go to the folder then
### `npm install`

create .env file in root deroctory and type your value for 
    DB_HOST=
    DB_PORT=
    DB_USERNAME=
    DB_PASSWORD=
    DB_DATABASE=
    ACCESS_TOKEN_SECRET_KEY=
    ACCESS_TOKEN_EXPIRE_TIME=

save the file.


To run type on terminal and press enter
### `npm run start:dev`
